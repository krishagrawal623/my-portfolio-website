## 🚀 Getting Started

Clone the repository:

```bash
git clone https://github.com/krishagrawal623/my-portfolio-website.git
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

---

## 📦 Build
